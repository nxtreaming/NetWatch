# NetWatch 代理配置示例文件
# 格式: IP:端口:类型:用户名:密码
# 类型支持: socks5, http
# 用户名和密码可选，如果代理不需要认证则省略

# SOCKS5 代理示例（无认证）
192.168.1.100:1080:socks5
10.0.0.1:1080:socks5
203.0.113.1:1080:socks5

# HTTP 代理示例（无认证）
192.168.1.101:8080:http
10.0.0.2:3128:http
203.0.113.2:8080:http

# SOCKS5 代理示例（需要认证）
192.168.1.102:1080:socks5:username:password
10.0.0.3:1080:socks5:user1:pass1
203.0.113.3:1080:socks5:myuser:mypass

# HTTP 代理示例（需要认证）
192.168.1.103:8080:http:username:password
10.0.0.4:3128:http:user2:pass2
203.0.113.4:8080:http:proxyuser:proxypass

# 更多示例
# 您可以添加多达5000+个代理配置
# 每行一个代理，格式保持一致

# 注意事项：
# 1. 以 # 开头的行为注释，会被忽略
# 2. 空行会被自动跳过
# 3. 格式错误的行会被跳过，不会影响其他行的导入
# 4. IP地址必须是有效的IPv4地址
# 5. 端口范围: 1-65535
# 6. 类型只支持 socks5 和 http（小写）
# 7. 用户名和密码不能包含冒号(:)字符
